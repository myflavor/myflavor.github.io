# 快速开始

通过阅读快速开始，你可以了解NoActive的基本功能

# 依赖环境

  • Android 12+
  • Magiskopen in new window或KernelSUopen in new window
  • Lsposedopen in new window

# 特殊准备

  • MIUI & HyperOS必须刷入Millet_Config_V2.zipopen in new window
  • ColorOS & RealmeUI必须刷入Hans_Config.zipopen in new window

# 模块安装

前往NoActive下载站open in new window下载最新的NoActive

下载站Pro文件夹为付费版，正式版最新V3.2已停止更新

打开LSPosed启用NoActive模块勾选系统框架

# 模块配置

打开NoActive，点击右上角设置，根据内核支持选择 ，以下是根据内核版本推荐的冻结方式

  • 5.0+ V2
  • 4.19 MIUI V1
  • 4.19 其它 Kill

# 模块生效

作用域系统框架的应用，启用后重启设备生效，重启后打开Lsposed管理器查看日志

# 模块交流

点击链接加入QQ频道【Android•墓碑】open in new window

# 付费使用

通过正式版设置的支付宝打赏入口，打赏后在频道反馈区发送打赏截图，等待一段时间后会给你添加付费后的身份组

身份组添加后在填写公告子频道的付费信息问卷，以便后续接收Pro版本的激活码，一码一机，不可多设备使用，不可转让
